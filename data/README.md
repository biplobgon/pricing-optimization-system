# Data Directory

Place the Kaggle retail pricing dataset at:

```text
data/raw/retail_price.csv
```

Expected development flow:

1. Keep original downloaded files in `data/raw/`.
2. Write cleaned datasets to `data/processed/`.
3. Write engineered feature tables to `data/features/`.
4. Write forecasts and pricing recommendations to `data/predictions/`.

The current `data/raw/retail_price.csv` is a tiny sample placeholder. Replace it with the real Kaggle file before modeling.
